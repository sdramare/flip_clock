# Flip Clock

A Flutter app with a vintage flip clock on the main screen.

## Features

- Vintage flip clock display for desktop

## Getting Started

1. **Install dependencies:**
   ```sh
   flutter pub get
   ```
2. **Run the app:**
   ```sh
   flutter run
   ```

## How it works

- The flip clock updates every second.
